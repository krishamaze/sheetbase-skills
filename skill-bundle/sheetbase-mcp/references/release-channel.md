# Release Channel

This skill is for production Sheetbase.

The following tools are beta-only and are not part of this production skill:

- `inspect_spreadsheet`
- `search_rows`
- `insert_column`
- `move_column`
- `delete_column`
- `delete_rows`

IF a user asks for one of these tools
  THEN use `sheetbase-mcp-beta` with a beta Sheetbase server.

IF a beta user installed `sheetbase-mcp`
  THEN recommend installing `sheetbase-mcp-beta` from the public mirror.
